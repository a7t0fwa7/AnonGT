___

> ### git clone https://github.com/gt0day/AnonGT
> ### cd AnonGT && sudo bash install.sh
> ### exit

## AnonGT Commands

___

### print banner help

> ### sudo anongt

### Anonymous Mode Start

> ### sudo anongt start

### Anonymous Mode Stop

> ### sudo anongt stop

### Anonymous Mode Change Tor IP

> ### sudo anongt chngid

### Watch Tor Traffic

> ### sudo anongt status

### Change Mac Address For All Interfaces

> ### sudo anongt chngmac

### Revert Mac Address For All Interfaces

> ### sudo anongt rvgmac

### Clear Configs & Logs

> ### sudo anongt wipe
